HTML INTRODUCTION


HTML Stands for Hyper Text Markup Language which is standard markup langauge for documents designed to be displayed in a wed browser.



HTML Elements tells the browser how to display the content in a browser.



Can be developed using TEXT Editors

HTML HEADING


HTML Headings are defined with <h1> to <h6> tags,where <h1> define the most important heading <h6>define the least important heading



To define the respective headings of your document.



<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>

HTML LINKS


HTML Links are defined using <a> tag



To Creates a hyperlink to web pages , addresses,locations in the same page.



<a href="url" >Some text</a> Example:<a href="https://quickprep.github.io>QUICKPREP</a> Where href is an attribute of a tag.

HTML IMAGES


Defined using <img> tag.



To hardcode images in your HTML Document.



<img src="url" alt="alternate_text">
Example:<img src="logo.jpg" alt="logo">
Where src and alt are the attributes of img tag.

HTML TABLES


HTML TABLE is defined using <table> tag.
Each TABLE Row is defined using <tr> tag.
Each TABLE Header is defined using <th> tag.
Each TABLE Cell is defined using <td> tag



To arrange the data in rows and columns.



<table> <tr> <th>Heading 1 </th> <th>Heading 2 </th> </tr> <tr> <th>Data 1 </th> <th>Data 2 </th> </tr> <tr> <td>Data 1 </td> <td>Data 2 </td> </tr> </table>

HTML COLSPAN AND ROWSPAN


Spanning cells of a table using attributes such as colspan and rowspan.



To make it flexible as per user requirements.



<th colspan="2">QUICKPREP</th> ->Spans over 2 columns <th rowspan=2">YOUR PLATFORM</th> ->Spans over 2 rows.

HTML LIST


HTML LIST is defined used <ul> or <ol> tag and <li> tag.
<ul>refers to unordered list.
<ol>refers to ordered list.
<li>refers to each list item.



To group a set of related items in lists.



<ul> <li>Home</li> <li>About</li> <li>Contact us</li> </ul> <ol> <li>Home</li> <li>About</li> <li>Contact us</li> </ol>

HTML DESCRIPTION LIST

HTML CLASS ATTRIBUTE


Can be defined using keyword class. The HTML class attribute is used to specify a class for an HTML element.



The class attribute is often used to point to a class name in a style sheet. It can also be used by a JavaScript to access and manipulate elements with the specific class name.



Using a keyword class as an attribute in any HTML element.
Example:
<h1 class="heading1">hai</h1>
<p class="heading1">hello</p>

HTML ID ATTRIBUTE


ID refers to identifier and can be defines using keyword id



Used to point the specific style declaration in a style sheet. Cannot have more than one element with the same id in an HTML Document.



<h1 id="identifier">Hai</h1>
