String methods in JavaScript


Difficulty Level : Easy

String methods help you to work with the strings.Find below them in detail.

The slice() Method


slice() extracts a part of a string and returns the extracted part in a new string.

The method takes 2 parameters: the start position, and the end position (end is not included).

The slice() method also accepts negative value, where -1 represents the last index

This below example slices out a portion of a string from position 5 to position 11 (12-1):


let str = "Shreeshantha IT Solutions";
str.slice(5, 12);
 //returns shantha

The replace() Method


The replace() method replaces a specified value with another value in a string:

The replace() method does not change the string it is called on. It returns a new string.

By default, the replace() method replaces only the first match and it is case sensitive


let str = "Shreeshantha it Solutions";
str.replace("it", "IT");
 //returns Shreeshantha IT Solutions

The trim() Method


The trim() method removes whitespace from both sides of a string:


let str = " Shreeshantha IT Solutions ";
str.trim();
 //returns Shreeshantha IT Solutions

The padStart() and padEnd() Method


The padStart()and padEnd() method support padding at the beginning and at the end of a string


let str = "6";
str.padStart(4,0);
 //returns 0006


let str = "6";
str.padEnd(4,0);
 //returns 6000

The charAt() Method


The charAt() method returns the character at a specified index (position) in a string:


let str = "Shreeshantha IT Solutions";
str.charAt(0);
 //returns S

The charCodeAt() Method


The charCodeAt() method returns the unicode of the character at a specified index in a string


let str = "Shreeshantha IT Solutions";
str.charCodeAt(0) ;
 //returns 83
