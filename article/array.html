Array methods in JavaScript


Difficulty Level : Easy

Array methods help you to work with the arrays.Find below them in detail.

The join() Method


The join() method is used to join all array elements into a string by a separator.

We can the specify the separator as shown below.


const lang=[“Html”,”Css”,”Js”];
lang.join(“ * “);
 //returns Html*Css*Js

The shift() Method


Shifting is equivalent to popping, working on the first element instead of the last.

The shift() method removes the first array element and "shifts" all other elements to a lower index.


const lang=[“Html”,”Css”,”Js”];
lang.shift();
// returns Css,Js.

The unshift() Method


The unshift() method adds a new element to an array at the beginning, and "unshifts" older elements:


const lang=[“Html”,”Css”,”Js”];
lang.unshift(“php” );
 //returns returns php,Html,Css,Js

The delete() method


The delete() method is used to delete the specific element from an array amd make it undefined


const lang=[“php”,“Html”,”Css”,”Js”];
delete lang[0];
//deletes "php" from an array and make it undefined

The splice() method


The splice() method is used to add new items to an array:

The first parameter defines the position where new elements should be added (spliced in).

The second parameter defines how many elements should be removed.

The rest of the parameters define the new elements to be added.


const lang=[“php”,“Html”,”Css”,”Js”];
lang.splice(2, 0, "java", "angular");
 //returns php,Html,Java,angular,Css,Js

The slice() method


The slice() method creates a new array. It does not remove any elements from the source array


const lang1 = ["Html", "Css", "Js", "php", "angular"];
const lang2 = lang1.slice(1);
console.log(lang2);
 //returns Css,Js,php,angular
