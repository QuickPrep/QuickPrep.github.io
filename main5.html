45.

int main()
{
    int a = 1; 
    printf("%d %d %d", a, ++a, a++); 
    return 0;
}
WHAT IS THE OUTPUT OF THE ABOVE SNIPPET?
1 1 1
2 2 1
3 3 1
4 4 1
                 

Output : 3 3 1
Explaination:__cdecl is the default calling function for C programs.
And Argument passing order for calling functions is from right to left.
Hence,a++ will be executed first,where 1 will be printed,and a gets 
updated to 2.
then,++a will be executed second,which is 3.
finally,a is executed ,which is 3 agian.
Hence, 3 3 1.

Know more about __Cdecl

49.

#include "stdio.h" 
int main() 
{
    int *ptr = NULL; 
    printf("%d",sizeof(ptr));
    return 0;
}
What is the output of the above code?
A.0
B.2
C.4
D.8

                 

Output : 8
Explaination:
The size of pointer is constant for all data types such as character, 
integer and even structures. The size of pointer depends on the 
machine we are using.For Example,
in a 32-bit machine, the pointer size is 4 bytes and
 in 64-bit machine, the pointer size is 8 bytes.
