14.What is the output of the following C snippet?
void main()
{
int c[ ]={2.8,3.4,4,6.7,5};
int j,*p=c,*q=c;
for(j=0;j<5;j++)
{
    printf(" %d ",*c);
    ++q;	
}
for(j=0;j<5;j++)
{
    printf(" %d ",*p);
    ++p;
}
}
 
                 

Output :2  2  2  2  2  2  3  4  6  5
Reason : Initially pointer c is assigned to both p and q.
In the first loop, since only q is incremented and not c ,
the value 2 will be printed 5 times.
In second loop p itself is incremented.
So the values 2 3 4 6 5 will be printed.

20.What will be the output of the following C code? 

int main()
{   
    int i=0;
    while ( +(+i--) != 0)
        i-=i++;
    printf("%d",i);
    return 0;  
 }
 
                 

Output : -1 
If we 1st consider the while condition unary + doesn't do anything so the simplified condition becomes (i–) != 0
So i will be compared with 0 and then decremented no matter whether condition is true or false.
Since i is initialized to 0, the condition of while will b false at the 1st iteration
itself but i will be decremented to -1.
The body of while loop will not be executed.
And finally printf will print -1.
